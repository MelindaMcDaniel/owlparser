DoORS: Domain Ontology Ranking System

Ontology Quality Assessment Module

This web application implements one module of an ontology ranking system. It uses the Semiotic Ontology Metrics Suite to evaluate an ontology on their Syntactic, Semantics, Pragmatics and Social quality. The ontology must be represented in the owl/xml format. The University of Manchester provides an excellent OWL syntax converter service if conversion is needed. Click here for Syntax Converter. An overall score for the ontology will be calculated based on the semiotic layers selected.

Layer Selection

Select the semiotic layers to be considered.
